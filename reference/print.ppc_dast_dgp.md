# Print Method for Internal DAST PPC Objects

Prints key posterior predictive discrepancy summaries returned by
[`ppc_dast_dgp()`](https://claudiofronterre.github.io/RiskMap/reference/ppc_dast_dgp.md).

## Usage

``` r
# S3 method for class 'ppc_dast_dgp'
print(x, ...)
```

## Arguments

- x:

  An object of class `"ppc_dast_dgp"`.

- ...:

  Not used.

## Value

Invisibly returns `x`.
