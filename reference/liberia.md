# River-blindness in Liberia

This data-set contains counts of reported onchocerciasis (or
riverblindess) cases from 91 villages sampled across Liberia. The
variables are as follows:

- lat latitude of the of sampled villages.

- long longitude of the sampled villages.

- ntest number of tested people for the presence nodules.

- npos number of people that tested positive for the presence of
  nodules.

- elevation the elevation in meters of the sampled village.

- log_elevation the log-transformed elevation in meters of the sampled
  village.

## Usage

``` r
data(liberia)
```

## Format

A data frame with 90 rows and 6 variables
